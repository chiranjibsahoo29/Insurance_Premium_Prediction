# 💰 Insurance Premium Category Prediction

A Machine Learning powered web application that predicts the **Insurance Premium Category** based on user details such as age, income, lifestyle, and occupation.

This project demonstrates **end-to-end ML deployment** using FastAPI and Streamlit.

## 🧠 Project Overview

The system takes user inputs and predicts the insurance premium category using a trained Machine Learning model.

### Input Features:
- Age
- Weight
- Height
- Annual Income (LPA)
- Smoker Status
- City
- Occupation

### Output:
- Predicted Premium Category
- Confidence Score
- Class Probabilities
